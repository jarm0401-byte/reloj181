# CLAUDE.md

## Proyecto
Portafolio web de la fotógrafa Reloj181, construido con HTML, CSS y JavaScript puros sin frameworks.

## Objetivo
Mostrar el trabajo de la fotógrafa en secciones de fotografía y video, con una navegación simple, estética visual y un formulario de contacto para agendar sesiones.

## Estructura propuesta
- /css: estilos del sitio
- /js: comportamiento interactivo básico
- /img: imágenes y recursos visuales

## Páginas previstas
- index.html: home y CV visual
- smash-cakes.html
- sesiones-personales.html
- cortometrajes.html
- comerciales.html
- agenda.html

## Requisitos
- Sin frameworks ni librerías externas
- Diseño responsive y accesible
- Navegación clara y coherente entre secciones
- Preparado para crecer con más contenido y galería visual

## Estado actual
Se ha iniciado la estructura base del sitio con la documentación y el esqueleto principal de la página de inicio.
